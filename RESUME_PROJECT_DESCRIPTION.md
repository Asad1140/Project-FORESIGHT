# RESUME PROJECT DESCRIPTION: PROJECT FORESIGHT
**Demand & Inventory Intelligence System for DTC Retail**

Below are three resume project descriptions of varying lengths and formats designed to highlight the technical achievements and business impact of Project FORESIGHT.

---

### 1. Two-Line Version (Ultra-Concise)
*   Engineered an end-to-end demand forecasting pipeline using a **Random Forest Regressor** and **TimeSeriesSplit** chronological validation, improving forecast accuracy by **13.60% WAPE** compared to the seasonal baseline.
*   Designed a Streamlit inventory dashboard mapping SKU risk quadrants and financial metrics, protecting bestseller revenues and releasing working capital locked in overstocks.

---

### 2. Four-Line Version (Standard Professional)
*   Developed a weekly demand forecasting machine learning system for DTC retail using **Python (Pandas, NumPy, Scikit-Learn)** and a **Random Forest Regressor** over a 6-week horizon.
*   Implemented an automated data engineering pipeline performing range checking, referential integrity checks, and **IQR-based outlier capping** to clean 90,000+ daily sales records.
*   Achieved a **9.86% WAPE** (Weighted Absolute Percentage Error) on the backtest dataset, representing a **13.60% improvement** in replenishment accuracy over the Seasonal Naive baseline.
*   Designed an interactive Streamlit inventory decision dashboard displaying Plotly visualizations, early-warning stockout/overstock risk scores, and SKU reorder recommendations.

---

### 3. ATS-Friendly Version (Keyword & Bullet-Point Optimized)
**DTC Inventory & Demand Forecasting Machine Learning System (Project FORESIGHT)**
*   **Machine Learning Engineering**: Built a weekly demand forecasting model using a **Random Forest Regressor** in **Python**, forecasting sales volumes over a 6-week horizon.
*   **Data Engineering & Preprocessing**: Developed an automated data cleaning and quality auditing module in **Pandas** and **NumPy** that runs deduplication, dtype validation, referential integrity verification, and **IQR (Interquartile Range) outlier capping** on 90,000+ transactions.
*   **Time-Series Validation**: Applied chronological **TimeSeriesSplit** cross-validation on historical demand series to prevent temporal leakage and evaluate model generalizability.
*   **Evaluation & KPI Verification**: Evaluated model accuracy against a Seasonal-Naive baseline using **WAPE (Weighted Absolute Percentage Error)**, **MAE**, **RMSE**, and **Forecast Bias**, achieving a **9.86% WAPE** and a **13.60% accuracy improvement**.
*   **Decision Intelligence & Visualization**: Programmed an interactive **Streamlit** dashboard using **Plotly** to visualize SKU risk matrix bubble grids, forecast confidence bands, and inventory gauges.
*   **Actionable Business Impact**: Built a supply chain risk engine classifying products into operational replenishment quadrants (`REORDER NOW`, `MARKDOWN / CLEAR`), protecting bestseller revenues and freeing up working capital.
